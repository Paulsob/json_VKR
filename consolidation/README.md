# Папка consolidation

Эта папка содержит файлы с информацией о маршрутах, используемых системой планирования расписаний водителей.

## route_numbers.txt

Файл `route_numbers.txt` содержит список номеров маршрутов, для которых будет производиться планирование. Каждый номер маршрута указывается на отдельной строке.

Пример содержимого:
```
9
20
21
47
48
55
61
```

При запуске планировщика система загружает эти номера маршрутов и создает файлы Расписание_Итог для каждого маршрута каждого дня в соответствующих подпапках.
